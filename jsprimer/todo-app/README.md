## Usage
実行環境にnpmが入っている場合
```sh
git clone git@github.com:gyokuro06/learning-js.git
cd jsprimer/todo-app
npm run dev
```

実行環境にnpmがなく、docker composeで動かす場合
```sh
git clone git@github.com:gyokuro06/learning-js.git
cd jsprimer/todo-app
make ci
// Ctrl-Cで終了する
make clean
```
